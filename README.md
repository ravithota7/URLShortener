# URLShortener
Url shortener app in spring boot

## To run locally
Pre-requisites
- JDK
- Maven
```
mvn clean package
```

## To run using docker
```
docker pull ravithota7/url-shortener-v1:latest
```
link to docker image [url-shortener](https://hub.docker.com/repository/docker/ravithota7/url-shortener-v1)
